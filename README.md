## 关于这个项目

这是一个个人研究网站，聚焦具身智能机器人导航领域（Embodied Intelligence Robot Navigation），同时探索独立开发者副业。

主要内容包括：

- **研究方向**：VLN视觉语言导航、ObjectNav目标导航、LLM导航、强化学习导航
- **论文库**：27篇核心论文，覆盖6大方向
- **博客**：研究日志、论文解读、创业思考
- **创业**：独立开发者实验项目和方法论
