物理基础化：VLA 时代 VLN 研究者的下一个主战场

2026-04-25 | 精卫导航 | 标签: VLA物理基础化VLN具身智能

2026年的具身智能领域有一个公开的秘密：VLA（Vision-Language-Action）模型在语义理解上已经超神，但在物理世界上仍是白痴。Figure 02 可以优雅地折叠衣物，π0 可以收拾桌面——但在真实家庭的厨房里，它们依然会撞上透明的玻璃门，踩空台阶，把陶瓷杯当成塑料杯用力捏。

这不是 Bug，是 Feature Gap：互联网规模的视觉-语言数据教会了模型"看到什么"，却没有教会模型"物理上怎么做"。

一、语义泛化 vs 物理理解：根本性鸿沟

VLA 的成功建立在 CLIP 类的语义表示上。CLIP 将"红色杯子"与"red cup"映射到高维空间的相近位置——这是语义泛化（Semantic Generalization）。但语义相近 ≠ 物理相同：

核心问题："拿起红色的杯子"
- 语义理解：找到图片中红色、与"杯子"外观相似的对象区域 ✓
- 物理理解：确定杯子的 3D 位置、材质（玻璃？陶瓷？塑料？）、重量等级、是否遮挡、是否有盖子 ✓
- 当前 SOTA：仅做到了语义理解，物理理解依赖手工编码的物理引擎 ✓

这个鸿沟在 VLN（视觉-语言导航）中尤为致命。当指令说"走到门旁边"——

方法"旁边"的理解物理验证
CLIP-based语义相似度 near❌ 无
3D 几何方法欧式距离 < 阈值✅ 碰撞检测
语言图方法拓扑关系 near❌ 无
FSCoT（本文）语义+几何双验证✅ 物理头约束

二、物理基础化的三条技术路线

路线1：CLIP 驱动的外观语义场

代表工作：LERF（ICRA 2023）、OpenScene（CVPR 2023）、SA3V（CVPR 2024）

核心思想：将 CLIP 的语义特征"浇注"到 3D 场景表示中，构建语言辐射场（Language Radiance Field）。给定文本查询，在 3D 空间中提取语义相关的区域。

优势：语义丰富，可以处理开放词汇查询

缺陷：完全基于外观相似度，不考虑物理属性。"玻璃门"和"透明塑料门"在 CLIP 空间中可能非常相似，但物理性质完全不同。

路线2：几何驱动的精确物理建模

代表工作：3D-MOT、PointNeXt、Uni3R

核心思想：用 3D 点云/TSDF 构建精确的度量地图，做碰撞检测、路径规划。

优势：物理上精确，几何误差可控

缺陷：缺乏语义理解——只知道"这里有个障碍"，不知道"这是可以推开的抽屉还是固定的书架"

路线3：功能驱动的物理属性预测

代表工作（新兴）：Physics-VLM、NPHY（CVPR 2026）

核心思想：用 VLM 预测对象的物理属性：材质类型、重量等级、可移动性、功能类别。

优势：同时建模语义和物理

缺陷：物理属性标注成本极高，域迁移差

三、为什么学术 VLN 研究者应该押注物理基础化

VLA 时代，"端到端学习"成了显学。但端到端 VLA 有两个致命问题：

第一，Scale 陷阱：大厂用数千块 H100 训练 VLA，学术团队没有这个资源。
第二，物理盲区：VLA 的预训练数据以 2D 图像-文本对为主，3D 物理先验严重不足。

物理基础化恰好避开了这两个陷阱：

  • 不需要 Scale：物理约束是"结构化先验"，不需要海量数据
  • 需要领域知识：刚体物理、接触力学、材质属性——这是垂类壁垒，不是大数据竞赛
  • 可以用小数据+强先验：牛顿力学 + 材质库 + 功能分类，可以让小模型比大 VLA 更懂物理

四、FSCoT 的物理基础化设计

本文提出的 FSCoT 框架在物理基础化上有系统性设计：

双头置信度机制

FSCoT 的语义头负责语言歧义消解（"哪个红色的杯子"），物理头负责几何可行性验证（"能否在不碰撞的情况下够到杯子"）。两个头的置信度独立计算，最终 STOP 决策由两者的联合置信度决定。

物理停靠约束（Physical Stop Constraint）

传统 VLN 的 STOP 决策只考虑"是否到达目标区域"。FSCoT 引入了物理停靠约束：

物理停靠 = 距离 < d_max ∧ 路径无碰撞 ∧ 视角可观测 ∧ 交互可达

这四个条件缺一不可。距离近但视角被遮挡 = 不能交互；无碰撞但距离过远 = 超出机械臂可达范围。

物理属性库的引入

FSCoT 引入了一个轻量级物理属性库，包含常见家居对象的：

  • 材质类型：玻璃 / 陶瓷 / 木材 / 塑料 / 金属
  • 重量等级：轻型（<500g）/ 中型（500g-2kg）/ 重型（>2kg）
  • 可移动性：静态 / 可推 / 可拉 / 可抓取
  • 交互方式：拿取 / 按压 / 推拉 / 旋转

这个库不需要昂贵的标注——可以从家具数据库（IKEA 物体库）和公开的材质数据集中自动构建。

五、2026 年的物理基础化研究前沿

以下几个方向值得关注：

  1. 触觉-视觉联合物理推理：Haptic-VLN 数据集（当前不存在，CVPR 2026 投稿窗口的空白机会）
  2. 视频预测+物理引擎：CoVeR、PhysDreamer 将视频生成与物理仿真结合，可以"想象"物理交互的结果
  3. 3D Gaussian + CLIP 的规模化：当 LERF 可以实时处理 1000 ㎡的真实家庭时，物理基础化的 VLN 就真的到来了
  4. 物理属性的大规模预训练：PhysNet 系列工作探索从视觉输入预测物理属性的通用能力

结论

VLA 时代，语义理解已经接近"解决"。下一个十年，具身智能的核心战场在物理基础化。

对于 VLN 研究者，这不是选择题，是必答题——不会物理的导航智能体，永远只能在模拟器里走直线。

VLA 做的是"看到就能说"，VLN+物理基础化要做的是"看到就能走通"。
