VLN停止决策：被所有人忽视的致命瓶颈

68%成功率的顶级VLN系统，停止误差平均2.5米。机器人"导航成功"了，却停在离目标5步远的地方——这不是bug，是系统性缺陷。

核心数字

HaltNav（2025）首次系统量化了VLN停止精度问题。结果比大家想象的更糟糕：

2.5m DUET平均停止误差
5× 超过机械臂操作范围
65% StoP指标（DUET基线）
3m SR"成功"的判定阈值

反直觉事实：VLN的主流评估指标SR（成功率），对"停在3米内"就算成功。但真正的机器人任务（拿东西、开门、充电）需要精确到0.5米以内。现有SOTA模型在"成功"的轨迹中，平均停在离目标2.5米的地方。

为什么STOP不是普通动作

VLN中有4类动作：前进、左转、右转、停止。现有模型（DUET、NavGPT等）把这4个动作放在同一个action vocabulary里，用完全相同的方式训练和预测。

这是根本错误。

❌ 方向动作（前进/转向）

依赖局部视觉线索：眼前有走廊就走进去，有门就考虑转向。只需要看当前帧。

✅ 停止动作（STOP）

需要全局语义对齐：当前位置的视觉特征 vs. 指令末端描述的目标。需要跨越整个轨迹做判断。

用一个例子说明：指令是"走到卧室门口，在床左边的椅子旁停下"。到达附近后，模型需要回答的问题是：

  • 我看到的椅子，是不是指令里说的那把？
  • 我站的位置，是不是"床左边"？
  • 现在停，还是再走一步？

这些问题没有一个能从当前帧的局部特征回答。模型需要把当前视觉场景和指令的目标描述做语义对齐，而这正是现有模型没有专门设计的能力。

StoP指标：量化停止质量

HaltNav提出了StoP（Stop Precision）指标，解决SR指标掩盖停止误差的问题。

# StoP@τ：在成功轨迹中，停止点在目标τ米内的比例 StoP@τ = |{i ∈ 成功轨迹 : d(stop_i, goal_i) ≤ τ}| / |成功轨迹| # 例子：τ=1.0m时，有多少成功轨迹的停止点在1米内？ # DUET baseline: StoP@1m ≈ 25%（4个成功里只有1个停在1米内） # 即使放宽到τ=3m（SR的阈值），StoP@3m ≈ 65%

可以画StoP曲线：横轴是阈值τ，纵轴是StoP@τ。曲线越陡峭、上升越快，停止精度越高。DUET的曲线在0.5~1.5m区间非常平缓，说明停止点分散在目标周围很大的范围内。

HaltNav的解决方案：RVH机制

HaltNav提出了Reactive Visual Halt（RVH）机制，核心思路：不改变模型的主体架构，而是在停止决策时引入额外的视觉比对模块。

RVH工作原理

  1. 构建Topological Memory：保留最近3个视觉观察帧（O_t-2, O_t-1, O_t）
  2. 提取Goal Reference：从指令中抽取目标描述子句（最后一个地点描述）
  3. 计算Visual-Semantic Similarity：当前帧视觉特征 vs. 目标描述的文本嵌入
  4. 设置Halt Threshold：相似度超过阈值τ=0.65时触发停止

RVH的局限：只用最近3帧，没有全局轨迹上下文；相似度阈值是静态的，无法适应不同难度的指令；最重要的是，没有专门针对停止决策的推理机制——仍然是"阈值触发"而不是"语义理解"。

实验结果

为什么停止误差在真实场景中被放大

学术benchmark用3米阈值，掩盖了这个问题的严重性。但真实机器人场景的要求完全不同：

  • 充电桩对接：需要精确到±5cm，3米停止误差=完全失败
  • 物品抓取：机械臂伸展范围约50cm，停止误差>50cm=无法完成任务
  • 开门操作：门把手高度固定，停止位置偏差>20cm=操作失败
  • 人机交互：服务机器人停在2.5米外递东西，用户体验极差

结论：现有VLN系统距离实际部署还有一个被严重低估的障碍——停止精度。不解决停止决策问题，SR从68%提升到80%只是数字游戏，真实有用性没有本质改变。

停止误差的根本原因分析

HaltNav的分析把停止错误分为3类：

Type A
过早停止（55%）
到达正确区域前就停了。主因：路径中遇到视觉相似的"干扰场景"（similar-looking rooms），模型误以为到了。
Type B
过晚停止（35%）
经过了目标位置没停，继续走直到迷路。主因：指令目标描述模糊，模型不确定时选择继续走（更保守）。
Type C
精度不足（10%）
停在了正确位置附近，但偏差超过1米。主因：缺乏精细的深度/距离感知，无法确定"刚好在这里"。

三种错误类型背后是同一个根本问题：停止决策缺乏专门的语义对齐机制。模型用和方向决策完全相同的特征做停止判断，但停止需要的是"当前场景与目标描述的匹配度"，而不是"下一步往哪走"。

研究价值：为什么这个方向值得做

停止决策是VLN走向真实部署的关键缺口，而且是个相对低竞争的方向。截至2025年Q1：

  • 专门研究停止决策的论文：<5篇
  • 提出停止精度指标的论文：只有HaltNav
  • VLN整体论文年产量：200+篇

大家都在卷SR，没人盯StoP。这是一个明显的研究gap，而且和工业落地直接相关。

TL;DR — 3个关键结论

  • STOP≠普通动作：它需要全局语义对齐，而现有模型没有为此专门设计
  • StoP指标揭示了被SR掩盖的系统性问题：DUET成功轨迹中75%的停止点超出1米
  • 停止精度是VLN落地的关键瓶颈，而且是个研究空白——方向正确，时机成熟
