# Juicero — $700物联网榨汁机的硅谷最大笑话

**融资$1.2亿、Google Ventures投资——用手挤压袋子出汁率更高**

---

## 公司概况

| 项目 | 信息 |
|------|------|
| 名称 | Juicero |
| 地点 | 美国加州 |
| 赛道 | 智能家电 / 食品科技 |
| 成立时间 | 2013年 |
| 总融资 | $1.18亿美元 |
| 关闭时间 | 2017年9月 |
| 估值峰值 | ~$4亿美元 |

---

## 公司背景

- 创始人：Doug Evans（连续创业者，曾创立灌装有机果汁公司）
- 核心产品：Juicero Press——一台价值$699的WiFi联网榨汁机
- 商业模式：用户购买 Juicero 机器后，需要订阅专用果汁包装袋（每包约$5-$10）
- 投资人：Google Ventures（Google Ventures首次在硬件领域的重大投资）、Kleiner Perkins、Campbell Soup等

---

## 失败原因分析

### 1. 核心问题：极度过度工程化

Juicero Press 的技术规格看起来极其先进：
- 4吨压力挤压（官方声称相当于"举起一辆法拉利"）
- WiFi 连接，可检测袋子新鲜度
- 配套App追踪每日果汁摄入

问题是：2016年《彭博商业周刊》记者做了一个简单测试——<strong>直接用手挤压那个专用的果汁袋，挤出同样多的汁液，比Juicero机器更快</strong>。

这台价值700美元的机器，做的事情和用手挤袋子的效果几乎一样。

### 2. 用户需求是真实存在的，但产品形态错了

用户确实需要方便的鲜榨果汁——这是一个真实需求。但Juicero错误地认为：
- 用户需要一个"高科技产品"来解决这个需求
- 实际上，用户只需要"新鲜好喝的果汁"，不需要一个会联网、会推送固件更新、占地一平方英尺的机器

### 3. 价格门槛摧毁了需求

$699的机器 + $5-10/包的订阅，对大多数消费者来说门槛太高。
- 一台普通榨汁机（Vitamix等）只要$200-400
- 一杯鲜榨果汁在店里买只要$8-12
- Juicero的每月订阅成本（$30-50）相当于每天去果汁吧

### 4. 投资者的"工程师偏见"

Google Ventures 和 Kleiner Perkins 的合伙人大多是技术背景出身，他们看到了 Juicero 的技术创新，却忽略了：
- 技术复杂 ≠ 用户价值高
- 工程师觉得"4吨压力"很酷，普通消费者只觉得"挤个果汁要这么麻烦？"

---

## 关键教训

### 对创业者的启示

1. **"技术精妙"和"用户需要"之间有一条巨大的鸿沟**
   - Juicero 的工程师花了大量时间优化压力传感器和 WiFi 模块
   - 但这些技术复杂度，用户完全感知不到
   - 在产品初期，应该把资源放在"用户能直接感受到的价值"上

2. **永远不要让投资人测试你的产品**
   - Juicero 的公关策略是"高大上的科技感"
   - 但当记者真的开始使用时，发现了这个产品的荒谬
   - 在产品推出前，让非技术背景的真实用户测试产品

3. **高价硬件产品需要极强的使用频次来摊薄成本**
   - $700的机器，如果用户每周用一次，需要使用1年才能摊薄硬件成本
   - 但大多数用户的使用频次远低于预期
   - 订阅模式的前提是用户真的每天在喝你的果汁

4. **最好的技术不是解决"没有问题的问题"，而是让"已存在的问题"解决得更优雅**
