# 宇树科技：王兴兴的硬核创业路

## 案例速览
- **创始人**：王兴兴（1990年生，浙江余姚）
- **创立时间**：2016年
- **赛道**：具身智能/四足机器人 → 人形机器人
- **最新融资**：拟IPO，IPO前估值127亿
- **身家**：67亿人民币（2026年）
- **市场地位**：全球四足机器人市占率近70%，人形机器人出货量全球第一

---

## 创业故事

### 偏科少年的逆袭
王兴兴的成长路径与传统精英截然不同。1990年出生于浙江余姚普通家庭，初中英语成绩极差，被老师认为"笨"，但动手能力极强，热衷制作微型涡轮喷气发动机。2006年中考险些未考上普通高中，英语仅及格3次，但数理化成绩优异。

2009年大学期间，他用**200元自制简陋双足机器人**，虽然行走不稳，但这是宇树科技的起点原型。

### 关键转折点

| 年份 | 事件 |
|------|------|
| 2013年 | 进入上海大学读研，在导师支持下研发四足机器人XDog，放弃主流液压方案，采用**电驱动技术**，成本控制在2万内 |
| 2015年 | 主动申请延期毕业完成XDog，在国际智能"星创师"大赛获二等奖，赢得8万奖金 |
| 2016年 | XDog视频网络走红，放弃大疆工作机会，在杭州创办宇树科技，获200万天使投资 |
| 2017年 | 首款产品莱卡狗获订单，面临量产困境，资金危机中自掏腰包维持运营 |
| 2018年 | 新资金注入，产品稳定生产，公司步入正轨 |
| 2021年 | 央视春晚"机器牛"舞蹈表演，完成**全球首次四足机器人集群表演** |
| 2025年 | 蛇年春晚表演《秧BOT》，机器人实现抛接手绢、侧空翻等高难度动作 |

---

## 成功关键

1. **技术偏执**：放弃大厂高薪，选择电驱动路线，避开波士顿动力液压方案
2. **成本 killer**：用200元造机器人，用低价切入消费级市场
3. **事件破圈**：春晚、冬奥、超级碗——大型活动是科技品牌破圈神器
4. **开源共创**：公开机器人数据集，推动行业协同创新

---

## 关键教训

- **"技术可行"≠"商业可用"**：硬科技创业要熬过死亡谷，核心是低成本快速迭代
- **创始人基因决定公司气质**：技术偏执者也能成功，偏科不是劣势是专注
- **品牌曝光是硬科技公司必修课**：技术牛 ≠ 市场赢，需要让大众"看见"
