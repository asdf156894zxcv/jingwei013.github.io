# 案例分析：Shape Robotics — 教育机器人先驱的陨落

**烧掉1500万美元、被疫情击垮的教育机器人公司**

---

## 公司概况

| 项目 | 信息 |
|------|------|
| 名称 | Shape Robotics |
| 地点 | 丹麦 |
| 赛道 | 教育机器人 (STEM Education) |
| 成立时间 | 2015年 |
| 总烧钱 | $15M (1500万美元) |
| 倒闭时间 | 2022年 |

---

## 失败原因分析

### 1. 硬件经济困境

- 高单位成本
- 复杂的供应链
- 教育市场销售周期长（学校采购流程复杂）

### 2. 市场竞争激烈

- 面临乐高教育等老牌厂商
- 新进入者的价格战
- 无法建立差异化壁垒

### 3. 市场时机不佳

- COVID-19疫情严重冲击分销渠道（学校）
- 线下教学停摆导致产品无法交付

### 4. 资本效率低下

- 未能实现可持续的单位经济效益
- 烧光资金后停止运营

---

## 关键教训

### 对创业者的启示

1. **硬件陷阱**
   - 纯硬件产品在教育领域需要10倍更优的单位经济
   - 或需要强大的软件杠杆（85%毛利率的SaaS模式）

2. **商业模式优先**
   - 应优先考虑软件驱动的SaaS模式
   - 自下而上的免费增值模式更适合教育市场

3. **抗风险能力**
   - 业务模式需要具备抗冲击性
   - 可考虑基于浏览器的模拟环境，减少对物理硬件的依赖
