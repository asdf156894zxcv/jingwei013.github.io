## 定位（用户明确）
- **主线**：博士研究 → 目标是生产ICRA 2027/RAL论文
- **论文模块**：参考论文库 + 我的论文（像VLN-TimeLine时间线）

## 新架构

### 导航栏（5个Tab）
1. **首页** - Hero + 研究方向卡片
2. **论文** - 时间线视图
3. **博客** - 合并后的博客列表
4. **关于** - 个人简介+联系

### 首页（研究方向）
- Hero：名字+身份+统计数字
- 4个研究方向卡片（CoW-Nav / EmbodiedSLAM / 世界模型 / VLA）
- 每个卡片展示：方向名、描述、当前进度、关键论文链接
- 底部：最新博客预览（3篇）

### 论文页（时间线）
参考VLN-TimeLine风格：
- 左侧：垂直时间轴（年份+月份）
- 右侧：论文卡片列表
- 两类论文：
  - 🏆 **我的论文**：已发表/在投论文（突出显示，有DOI/arXiv链接）
  - 📚 **参考文献**：按研究方向分类的相关论文
- 按年份分组：2026 / 2025 / 2024 / 更早
- 筛选：全部 / 我的论文 / 参考文献 / 按方向（CoW-Nav / VLN / SLAM / VLA）

### 博客页
- 合并原博客 + 原案例
- 筛选标签：全部 / 研究笔记 / 创业思考 / 热点观察 / 日记
- 按月份归档
- 博客内容：技术深度博客（保留）+ 案例分析（保留）+ 思考日记（新增）
